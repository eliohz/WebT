Linux Permission Calculator

Information Eingabe und Ausgabe

Dieser Linux-Berechtigungsrechner wurde von Elio Heinz für des Moduls "Web Technologies" als praktische Arbeit im HS24 entwickelt.
User können Linux-Dateiberechtigungen einfach verwalten, indem sie zwischen symbolischer Darstellung, numerischer Darstellung und Checkboxen wechseln.
Das Ergebnis wird automatisch in den anderen Feldern angezeigt.

  • Symbolische Darstellung: z. B. rwxrwxrwx
  • Numerische Darstellung: z. B. 777
  • Checkboxen: für detaillierte Anpassung der Berechtigungen
